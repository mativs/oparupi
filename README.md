oparupi
=======

Oparupi

Requirements
===========



Configure Vagrant
=================

    vagrant init precise32 http://files.vagrantup.com/precise32.box

Dependencies
============

- fabric
- virtualenv
- python
- libjpeg62 
- libjpeg62-dev 
- zlib1g-dev


Setup
============

    virtualenv --no-site-packages 

Setup Development Environment
=============================

    fab setup
